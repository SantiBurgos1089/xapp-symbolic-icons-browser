# xapp-symbolic-icons-browser
Unofficial application that shows all xapp-symbolic-icons available

This project is not affiliated nor associated or related to Linux Mint, the XApp project or any related group, this is a personal attempt into showing all the xapp-symbolic-icons (xsi) similar to the app gtk3-icon-browser, since there is no app available for that task.

Feel free to contribute, fork and check

## Technologies used
* libAdapta (XApp project fork of libAdwaita)
* GTK4
* Python 3
